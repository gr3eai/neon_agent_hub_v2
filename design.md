# تصميم تطبيق Neon Agent Hub V2

## نظرة عامة
منصة ذكية متقدمة لإدارة الوكلاء والمحادثات مع دعم متعدد المزودين (DeepSeek, Groq, OpenAI, Together).

## الشاشات الرئيسية

### 1. الشاشة الرئيسية (Home)
**المحتوى:**
- قائمة بالمحادثات الحديثة
- شريط بحث للبحث في المحادثات
- أزرار سريعة لإنشاء محادثة جديدة
- عرض الوكلاء المتاحين

**الوظائف:**
- عرض المحادثات المحفوظة محلياً
- البحث في المحادثات
- حذف المحادثات
- بدء محادثة جديدة

### 2. شاشة المحادثة (Chat)
**المحتوى:**
- رسائل المحادثة (المستخدم والوكيل)
- حقل إدخال النص
- زر الإرسال
- اختيار المزود (DeepSeek, Groq, OpenAI, Together)

**الوظائف:**
- إرسال الرسائل
- استقبال الردود من الوكيل
- تبديل المزود
- حفظ المحادثة تلقائياً

### 3. شاشة الأدوات (Tools)
**المحتوى:**
- قائمة بالأدوات المتاحة
- أيقونات للأدوات
- وصف مختصر لكل أداة

**الوظائف:**
- عرض الأدوات المتاحة
- الوصول السريع للأدوات
- إدارة الأدوات المفضلة

### 4. شاشة الإعدادات (Settings)
**المحتوى:**
- إدخال مفاتيح API للمزودين
- إدارة الأذونات
- خيارات التطبيق العامة
- معلومات التطبيق

**الوظائف:**
- حفظ مفاتيح API
- اختبار الاتصال بالمزودين
- إدارة الأذونات
- تخصيص الإعدادات

## التدفقات الرئيسية

### تدفق بدء محادثة جديدة:
1. المستخدم يضغط على زر "محادثة جديدة" في الشاشة الرئيسية
2. ينتقل إلى شاشة المحادثة
3. يختار المزود المفضل
4. يكتب رسالته ويضغط إرسال
5. يستقبل الرد من الوكيل
6. تُحفظ المحادثة تلقائياً

### تدفق إعداد مفاتيح API:
1. المستخدم يفتح التطبيق لأول مرة
2. ينتقل إلى الإعدادات
3. يدخل مفاتيح API للمزودين
4. يختبر الاتصال
5. يحفظ الإعدادات

### تدفق إدارة الأذونات:
1. المستخدم يذهب إلى الإعدادات → الأذونات
2. يختار الأذونات المطلوبة
3. يضغط على "طلب جميع الأذونات"
4. يوافق على الأذونات في نظام الأندرويد

## نظام الألوان

**الوضع الفاتح:**
- الخلفية الرئيسية: `#ffffff`
- الخلفية الثانوية: `#f5f5f5`
- النص الرئيسي: `#11181C`
- النص الثانوي: `#687076`
- اللون الأساسي (Primary): `#FFD700` (ذهبي)
- اللون الثانوي: `#1a1a1a` (أسود)

**الوضع الداكن:**
- الخلفية الرئيسية: `#0a0a0a` (أسود)
- الخلفية الثانوية: `#1a1a1a`
- النص الرئيسي: `#ECEDEE`
- النص الثانوي: `#9BA1A6`
- اللون الأساسي (Primary): `#FFD700` (ذهبي)
- اللون الثانوي: `#2a2a2a`

## التصميم السيبراني الحديث

- استخدام الألوان الذهبية والسوداء
- حواف مستديرة للبطاقات والأزرار
- ظلال خفيفة للعناصر
- أيقونات حديثة وواضحة
- تأثيرات انتقالية سلسة
- تصميم متجاوب لجميع أحجام الشاشات

## الأذونات المطلوبة

- الكاميرا (للتقاط الصور)
- الميكروفون (لتسجيل الصوت)
- الموقع (لتحديد الموقع الجغرافي)
- جهات الاتصال (للوصول لقائمة الاتصالات)
- الوسائط (للوصول للصور والفيديو)
- أجهزة استشعار الجسم (لقراءة بيانات الصحة)
- الأجهزة المجاورة (للاتصال بالأجهزة القريبة)
- معلومات الهاتف (للوصول لرقم الهاتف)
- الرسائل النصية (لإرسال واستقبال الرسائل)
